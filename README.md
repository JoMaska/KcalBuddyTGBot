# KcalBuddy 🍏
KcalBuddy — это телеграм бот для отслеживания потребления калорий, белков, жиров и углеводов.
С KcalBuddy вы сможете легко следить за тем, что вы едите, настроить свои цели по питанию и получить советы для достижения более здорового образа жизни!

## Связь с ботом

Чтобы получить доступ ко всем функциям, просто перейдите по ссылке на бота в Telegram: [@kcal_buddy_bot](https://t.me/kcal_buddy_bot).
